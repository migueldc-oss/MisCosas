## 简介

MisCosas 是一款库存管理应用，帮助您追踪自己的物品。您可以添加带有名称、价格、照片、描述的物品，并将它们组织到颜色编码的分组中。

**适用设备：** iPhone、iPad、Mac 和 Apple Vision Pro。

---

## 开始使用

首次打开应用时，您将看到一个空白的主屏幕。

![Main screen empty state](screenshots/empty-state.png)

点击 **+** 按钮添加您的第一个物品。

---

## 主屏幕

主屏幕以 **列表** 或 **网格** 视图显示您的所有物品。

![Main screen with items](screenshots/main-screen.png)

### 工具栏

顶部的工具栏提供对所有功能的快速访问：

| 按钮 | 操作 |
|---|---|
| 📁 | 分组 — 按分组筛选或管理分组 |
| ☰/▦ | 在列表视图和网格视图之间切换 |
| 🔽 | 排序菜单 — 按名称、价格或日期排序 |
| ⭐ | 切换收藏筛选 |
| ➕ | 添加新物品 |
| ⚙️ | 打开设置 |

### 筛选栏

当分组或收藏筛选处于活动状态时，工具栏下方会出现一个栏，显示当前筛选及一个 **✕** 按钮以清除筛选。

![Filter bar](screenshots/filter-bar.png)

## 管理物品

### 添加物品

1. 点击工具栏中的 **+** 按钮。
2. 填写物品详情：
   - **名称**（必填）
   - **价格**
   - **图片** — 点击从您的图库中选择照片
   - **颜色** — 选择颜色标签
   - **分组** — 分配到现有分组
   - **简短描述**
   - **详细描述**
3. 点击 **保存**。

![Add item form](screenshots/add-item.png)

### 查看物品详情

点击列表或网格中的任意物品以查看其完整详情。

![Item detail view](screenshots/item-detail.png)

详情屏幕显示：
- 全尺寸图片
- 价格
- 分组名称
- 创建和修改日期
- 描述

### 编辑物品

1. 打开物品详情视图。
2. 点击工具栏中的 **铅笔** 图标。
3. 修改字段。
4. 点击 **保存**。

### 删除物品

- **在 iPhone/iPad 上：** 在列表视图中向左滑动物品，然后点击 **删除**。
- **在 Mac 上：** 右键点击物品，然后选择 **删除**。

### 复制物品

- **在 iPhone/iPad 上：** 向左滑动物品，然后点击 **复制**。
- **在 Mac 上：** 右键点击物品，然后选择 **复制**。

## 管理分组

分组帮助您将物品组织到不同类别中。

### 创建分组

1. 点击工具栏中的 **📁** 按钮。
2. 选择 **管理分组**。
3. 点击右上角的 **+** 按钮。
4. 输入名称并点击 **保存**。

![Manage groups](screenshots/manage-groups.png)

### 重命名分组

1. 前往 **管理分组**。
2. 点击您想要重命名的分组。
3. 编辑名称并点击 **保存**。

### 删除分组

1. 前往 **管理分组**。
2. 向左滑动分组并点击 **删除**。

> **注意：** 删除分组 **不会** 删除其中的物品。物品将变为未分组状态。

### 按分组筛选

点击工具栏中的 **📁** 按钮并从菜单中选择一个分组。将仅显示该分组中的物品。

## 搜索和筛选

### 搜索

下拉物品列表以显示搜索栏。输入文本以按名称筛选物品。

![Search bar](screenshots/search.png)

### 收藏

点击工具栏中的 **⭐** 按钮以仅显示收藏物品。再次点击以显示所有物品。

### 排序

点击 **🔽** 按钮以排序物品：
- **名称**（A–Z）
- **价格**（从低到高）
- **日期**（最新的在前）

## 设置

点击 **⚙️** 按钮打开设置。

![Settings](screenshots/settings.png)

### 外观

选择 **系统**、**浅色** 或 **深色** 主题。

### 默认视图

选择物品默认以 **列表** 还是 **网格** 视图打开。

### 显示选项

- **显示价格** — 打开/关闭价格显示
- **图片大小** — 通过滑块调整缩略图大小

### 语言

从可用选项中选择应用语言。界面将立即更新。

![Language settings](screenshots/settings-language.png)

### 货币

选择价格显示和输入的货币。

![Currency settings](screenshots/settings-currency.png)

### 数据管理

- **恢复数据** — 用于未来备份恢复的占位符

---

## 提示与技巧

- **快速收藏：** 在列表视图中向右滑动物品以切换收藏星标。
- **快速分组筛选：** 使用工具栏中的 **📁** 菜单代替管理分组以进行快速筛选。
- **大图片：** 照片会自动调整大小以优化存储和同步。
- **iPad 和 Mac：** 左侧的边栏让您可以在分组和收藏之间快速切换。
- **Vision Pro：** 应用会自动适应 visionOS 界面。
