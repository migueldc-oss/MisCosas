## Introduction

MisCosas is an inventory management app that helps you keep track of your belongings. You can add items with names, prices, photos, descriptions, and organize them into color-coded groups.

**Available on:** iPhone, iPad, Mac, and Apple Vision Pro.

---

## Getting Started

When you first open the app, you'll see an empty main screen.

![Main screen empty state](screenshots/empty-state.png)

Tap the **+** button to add your first item.

---

## The Main Screen

The main screen shows all your items in either **list** or **grid** view.

![Main screen with items](screenshots/main-screen.png)

### Toolbar

The toolbar at the top provides quick access to all features:

| Button | Action |
|---|---|
| 📁 | Groups — filter by group or manage groups |
| ☰/▦ | Toggle between list and grid view |
| 🔽 | Sort menu — sort by name, price, or date |
| ⭐ | Toggle favorites filter |
| ➕ | Add a new item |
| ⚙️ | Open Settings |

### Filter Bar

When a group or favorites filter is active, a bar appears below the toolbar showing the active filter with an **✕** button to clear it.

![Filter bar](screenshots/filter-bar.png)

## Managing Items

### Adding an Item

1. Tap the **+** button in the toolbar.
2. Fill in the item details:
   - **Name** (required)
   - **Price**
   - **Image** — tap to select a photo from your library
   - **Color** — choose a color tag
   - **Group** — assign to an existing group
   - **Short description**
   - **Long description**
3. Tap **Save**.

![Add item form](screenshots/add-item.png)

### Viewing Item Details

Tap any item in the list or grid to see its full details.

![Item detail view](screenshots/item-detail.png)

The detail screen shows:
- Full-size image
- Price
- Group name
- Creation and modification dates
- Descriptions

### Editing an Item

1. Open the item detail view.
2. Tap the **pencil** icon in the toolbar.
3. Modify the fields.
4. Tap **Save**.

### Deleting an Item

- **On iPhone/iPad:** Swipe left on the item in list view and tap **Delete**.
- **On Mac:** Right-click the item and select **Delete**.

### Duplicating an Item

- **On iPhone/iPad:** Swipe left on the item and tap **Duplicate**.
- **On Mac:** Right-click the item and select **Duplicate**.

## Managing Groups

Groups help you organize your items into categories.

### Creating a Group

1. Tap the **📁** button in the toolbar.
2. Select **Manage Groups**.
3. Tap the **+** button in the top-right corner.
4. Enter a name and tap **Save**.

![Manage groups](screenshots/manage-groups.png)

### Renaming a Group

1. Go to **Manage Groups**.
2. Tap the group you want to rename.
3. Edit the name and tap **Save**.

### Deleting a Group

1. Go to **Manage Groups**.
2. Swipe left on the group and tap **Delete**.

> **Note:** Deleting a group does **not** delete the items inside it. Items become ungrouped.

### Filtering by Group

Tap the **📁** button in the toolbar and select a group from the menu. Only items in that group will be displayed.

## Searching and Filtering

### Search

Pull down on the item list to reveal the search bar. Type any text to filter items by name.

![Search bar](screenshots/search.png)

### Favorites

Tap the **⭐** button in the toolbar to show only your favorite items. Tap again to show all items.

### Sort

Tap the **🔽** button to sort items by:
- **Name** (A–Z)
- **Price** (low to high)
- **Date** (newest first)

## Settings

Open Settings by tapping the **⚙️** button.

![Settings](screenshots/settings.png)

### Appearance

Choose between **System**, **Light**, or **Dark** theme.

### Default View

Choose whether items open in **List** or **Grid** view by default.

### Display Options

- **Show Prices** — toggle price display on/off
- **Image Size** — adjust thumbnail size with a slider

### Language

Select the app language from the available options. The interface will update immediately.

![Language settings](screenshots/settings-language.png)

### Currency

Choose the currency for price display and input.

![Currency settings](screenshots/settings-currency.png)

### Data Management

- **Restore Data** — placeholder for future backup restoration

## Tips and Tricks

- **Quick favorite:** Swipe right on an item in list view to toggle the favorite star.
- **Fast group filter:** Use the **📁** menu in the toolbar instead of Manage Groups for quick filtering.
- **Large images:** Photos are automatically resized for optimal storage and sync.
- **iPad and Mac:** The sidebar on the left lets you quickly switch between groups and favorites.
- **Vision Pro:** The app adapts to the visionOS interface automatically.
