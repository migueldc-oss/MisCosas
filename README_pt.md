## Introdução

MisCosas é um aplicativo de gerenciamento de inventário que ajuda você a manter o controle de seus pertences. Você pode adicionar itens com nomes, preços, fotos, descrições e organizá-los em grupos codificados por cores.

**Disponível em:** iPhone, iPad, Mac e Apple Vision Pro.

---

## Primeiros Passos

Ao abrir o aplicativo pela primeira vez, você verá uma tela principal vazia.

![Main screen empty state](screenshots/empty-state.png)

Toque no botão **+** para adicionar seu primeiro item.

---

## A Tela Principal

A tela principal mostra todos os seus itens na visualização de **lista** ou **grade**.

![Main screen with items](screenshots/main-screen.png)

### Barra de Ferramentas

A barra de ferramentas na parte superior fornece acesso rápido a todas as funções:

| Botão | Ação |
|---|---|
| 📁 | Grupos — filtrar por grupo ou gerenciar grupos |
| ☰/▦ | Alternar entre visualização de lista e grade |
| 🔽 | Menu de ordenação — ordenar por nome, preço ou data |
| ⭐ | Alternar filtro de favoritos |
| ➕ | Adicionar novo item |
| ⚙️ | Abrir Configurações |

### Barra de Filtros

Quando um filtro de grupo ou favoritos está ativo, uma barra aparece abaixo da barra de ferramentas mostrando o filtro ativo com um botão **✕** para limpá-lo.

![Filter bar](screenshots/filter-bar.png)

## Gerenciar Itens

### Adicionar um Item

1. Toque no botão **+** na barra de ferramentas.
2. Preencha os detalhes do item:
   - **Nome** (obrigatório)
   - **Preço**
   - **Imagem** — toque para selecionar uma foto da sua biblioteca
   - **Cor** — escolha uma etiqueta de cor
   - **Grupo** — atribuir a um grupo existente
   - **Descrição curta**
   - **Descrição longa**
3. Toque em **Salvar**.

![Add item form](screenshots/add-item.png)

### Visualizar Detalhes do Item

Toque em qualquer item na lista ou grade para ver seus detalhes completos.

![Item detail view](screenshots/item-detail.png)

A tela de detalhes mostra:
- Imagem em tamanho real
- Preço
- Nome do grupo
- Datas de criação e modificação
- Descrições

### Editar um Item

1. Abra a visualização de detalhes do item.
2. Toque no ícone do **lápis** na barra de ferramentas.
3. Modifique os campos.
4. Toque em **Salvar**.

### Excluir um Item

- **No iPhone/iPad:** Deslize para a esquerda no item na visualização de lista e toque em **Excluir**.
- **No Mac:** Clique com o botão direito no item e selecione **Excluir**.

### Duplicar um Item

- **No iPhone/iPad:** Deslize para a esquerda no item e toque em **Duplicar**.
- **No Mac:** Clique com o botão direito no item e selecione **Duplicar**.

## Gerenciar Grupos

Os grupos ajudam você a organizar seus itens em categorias.

### Criar um Grupo

1. Toque no botão **📁** na barra de ferramentas.
2. Selecione **Gerenciar Grupos**.
3. Toque no botão **+** no canto superior direito.
4. Insira um nome e toque em **Salvar**.

![Manage groups](screenshots/manage-groups.png)

### Renomear um Grupo

1. Vá para **Gerenciar Grupos**.
2. Toque no grupo que deseja renomear.
3. Edite o nome e toque em **Salvar**.

### Excluir um Grupo

1. Vá para **Gerenciar Grupos**.
2. Deslize para a esquerda no grupo e toque em **Excluir**.

> **Nota:** Excluir um grupo **não** exclui os itens dentro dele. Os itens ficam sem grupo.

### Filtrar por Grupo

Toque no botão **📁** na barra de ferramentas e selecione um grupo no menu. Somente os itens desse grupo serão exibidos.

## Pesquisar e Filtrar

### Pesquisar

Puxe para baixo na lista de itens para revelar a barra de pesquisa. Digite qualquer texto para filtrar os itens por nome.

![Search bar](screenshots/search.png)

### Favoritos

Toque no botão **⭐** na barra de ferramentas para mostrar apenas seus itens favoritos. Toque novamente para mostrar todos os itens.

### Ordenar

Toque no botão **🔽** para ordenar os itens por:
- **Nome** (A–Z)
- **Preço** (do menor para o maior)
- **Data** (mais recente primeiro)

## Configurações

Abra Configurações tocando no botão **⚙️**.

![Settings](screenshots/settings.png)

### Aparência

Escolha entre o tema **Sistema**, **Claro** ou **Escuro**.

### Visualização Padrão

Escolha se os itens abrem na visualização de **Lista** ou **Grade** por padrão.

### Opções de Exibição

- **Mostrar Preços** — ativar/desativar a exibição de preços
- **Tamanho da Imagem** — ajustar o tamanho das miniaturas com um controle deslizante

### Idioma

Selecione o idioma do aplicativo entre as opções disponíveis. A interface será atualizada imediatamente.

![Language settings](screenshots/settings-language.png)

### Moeda

Escolha a moeda para exibição e entrada de preços.

![Currency settings](screenshots/settings-currency.png)

### Gerenciamento de Dados

- **Restaurar Dados** — placeholder para futura restauração de backup

## Dicas e Truques

- **Favorito rápido:** Deslize para a direita em um item na visualização de lista para alternar a estrela de favorito.
- **Filtro rápido de grupo:** Use o menu **📁** na barra de ferramentas em vez de Gerenciar Grupos para filtragem rápida.
- **Imagens grandes:** As fotos são redimensionadas automaticamente para armazenamento e sincronização otimizados.
- **iPad e Mac:** A barra lateral à esquerda permite alternar rapidamente entre grupos e favoritos.
- **Vision Pro:** O aplicativo se adapta automaticamente à interface visionOS.
