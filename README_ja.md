## はじめに

MisCosasは、持ち物を管理するための在庫管理アプリです。名前、価格、写真、説明を添えてアイテムを追加し、色分けされたグループに整理できます。

**対応デバイス：** iPhone、iPad、Mac、Apple Vision Pro

---

## 最初の手順

アプリを初めて開くと、空のメイン画面が表示されます。

![Main screen empty state](screenshots/empty-state.png)

**+** ボタンをタップして、最初のアイテムを追加します。

---

## メイン画面

メイン画面には、すべてのアイテムが **リスト** または **グリッド** 表示で表示されます。

![Main screen with items](screenshots/main-screen.png)

### ツールバー

上部のツールバーからすべての機能に素早くアクセスできます。

| ボタン | 操作 |
|---|---|
| 📁 | グループ — グループでフィルターまたはグループを管理 |
| ☰/▦ | リスト表示とグリッド表示を切り替え |
| 🔽 | 並べ替えメニュー — 名前、価格、日付で並べ替え |
| ⭐ | お気に入りフィルターを切り替え |
| ➕ | 新しいアイテムを追加 |
| ⚙️ | 設定を開く |

### フィルターバー

グループまたはお気に入りのフィルターが有効な場合、ツールバーの下にアクティブなフィルターを表示するバーが表示され、**✕** ボタンでクリアできます。

![Filter bar](screenshots/filter-bar.png)

## アイテムの管理

### アイテムの追加

1. ツールバーの **+** ボタンをタップします。
2. アイテムの詳細を入力します：
   - **名前**（必須）
   - **価格**
   - **画像** — タップしてライブラリから写真を選択
   - **色** — カラータグを選択
   - **グループ** — 既存のグループに割り当て
   - **短い説明**
   - **長い説明**
3. **保存** をタップします。

![Add item form](screenshots/add-item.png)

### アイテム詳細の表示

リストまたはグリッド内の任意のアイテムをタップすると、完全な詳細が表示されます。

![Item detail view](screenshots/item-detail.png)

詳細画面には以下が表示されます：
- フルサイズの画像
- 価格
- グループ名
- 作成日と更新日
- 説明

### アイテムの編集

1. アイテム詳細ビューを開きます。
2. ツールバーの **鉛筆** アイコンをタップします。
3. フィールドを変更します。
4. **保存** をタップします。

### アイテムの削除

- **iPhone/iPadの場合：** リスト表示でアイテムを左にスワイプして **削除** をタップします。
- **Macの場合：** アイテムを右クリックして **削除** を選択します。

### アイテムの複製

- **iPhone/iPadの場合：** アイテムを左にスワイプして **複製** をタップします。
- **Macの場合：** アイテムを右クリックして **複製** を選択します。

## グループの管理

グループを使用すると、アイテムをカテゴリに整理できます。

### グループの作成

1. ツールバーの **📁** ボタンをタップします。
2. **グループを管理** を選択します。
3. 右上隅の **+** ボタンをタップします。
4. 名前を入力して **保存** をタップします。

![Manage groups](screenshots/manage-groups.png)

### グループの名前変更

1. **グループを管理** に移動します。
2. 名前を変更するグループをタップします。
3. 名前を編集して **保存** をタップします。

### グループの削除

1. **グループを管理** に移動します。
2. グループを左にスワイプして **削除** をタップします。

> **注意：** グループを削除しても、その中のアイテムは **削除されません**。アイテムはグループなしになります。

### グループでフィルター

ツールバーの **📁** ボタンをタップし、メニューからグループを選択します。そのグループのアイテムのみが表示されます。

## 検索とフィルター

### 検索

アイテムリストを下に引いて検索バーを表示します。テキストを入力してアイテムを名前でフィルターします。

![Search bar](screenshots/search.png)

### お気に入り

ツールバーの **⭐** ボタンをタップすると、お気に入りのアイテムのみが表示されます。もう一度タップすると、すべてのアイテムが表示されます。

### 並べ替え

**🔽** ボタンをタップして、アイテムを以下の順に並べ替えます：
- **名前**（A–Z）
- **価格**（安い順）
- **日付**（新しい順）

## 設定

**⚙️** ボタンをタップして設定を開きます。

![Settings](screenshots/settings.png)

### 外観

**システム**、**ライト**、**ダーク** のテーマから選択します。

### デフォルト表示

アイテムがデフォルトで **リスト** または **グリッド** 表示のどちらで開くかを選択します。

### 表示オプション

- **価格を表示** — 価格表示のオン/オフを切り替え
- **画像サイズ** — スライダーでサムネイルサイズを調整

### 言語

利用可能なオプションからアプリの言語を選択します。インターフェースはすぐに更新されます。

![Language settings](screenshots/settings-language.png)

### 通貨

価格表示と入力のための通貨を選択します。

![Currency settings](screenshots/settings-currency.png)

### データ管理

- **データを復元** — 今後のバックアップ復元用のプレースホルダー

## ヒントとコツ

- **クイックお気に入り：** リスト表示でアイテムを右にスワイプして、お気に入りの星マークをオン/オフします。
- **クイックグループフィルター：** すばやくフィルターするには、グループ管理の代わりにツールバーの **📁** メニューを使用します。
- **大きな画像：** 写真はストレージと同期を最適化するために自動的にリサイズされます。
- **iPadとMac：** 左側のサイドバーからグループとお気に入りをすばやく切り替えられます。
- **Vision Pro：** アプリはvisionOSインターフェースに自動的に適応します。
